# Job Market Intelligence System (SQL Project)

## Overview
This project analyzes job market data using SQL to identify high-demand and high-paying skills.

## Tools Used
- SQL (SQLite)
- DB Browser for SQLite

## Analysis Performed
- Top Paying Jobs
- Most In-Demand Skills
- Highest Paying Skills
- Skill Opportunity Score

## Dataset
The dataset is sourced from Kaggle:
https://www.kaggle.com/datasets/arshkon/linkedin-job-postings

Due to GitHub file size limitations, the dataset is not included in this repository.

## Key Outcome
This project identifies the most valuable skills by combining demand and salary insights.
